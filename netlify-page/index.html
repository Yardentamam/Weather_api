Weather API Documentation

Welcome to the Weather API documentation. This API provides simple and fast access to current weather conditions.

Getting Started

To use this API, you need to install the package and start the server:

npm install
npm start

API Endpoints

GET /api/weather

Get weather by city name or coordinates.

Parameters:

  • city - City name (e.g., "London")
  • lat - Latitude
  • lon - Longitude

Example Usage

curl "http://localhost:3000/api/weather?city=London"

Response Format

{
  "city": "London",
  "temperature": 15.5,
  "description": "Partly cloudy",
  "humidity": 65
}

Advanced Configuration

For production use, configure your environment variables in the .env file.

Security Notes

This API is designed for development and testing. For production deployments, ensure proper authentication and rate limiting.
