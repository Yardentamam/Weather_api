# Weather API

A simple, fast, and reliable REST API for fetching current weather conditions by location.

## Installation

```bash
git clone https://github.com/Yardentamam/Weather_api.git
cd Weather_api
npm install
```

## Usage

```bash
npm start
```

Then visit: http://localhost:3000/api/weather?city=London

## Features

- Get weather by city name
- Get weather by coordinates
- Mock data support (no API key needed for testing)
